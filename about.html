ABOUT THE
PROJECT

This project presents an AI-based pipeline for underwater video enhancement and maritime object detection. It is designed to improve the visibility of underwater frames before applying object detection, making it more suitable for challenging marine environments where clarity is poor and threats may be difficult to identify.

02
Enhancement Rationale

WHY ENHANCEMENT IS IMPORTANT

Direct object detection on raw underwater images often performs poorly because of low contrast, haze, blue-green color dominance, and distortion. By enhancing the image first, the detector receives cleaner and more informative frames, which improves downstream detection performance.

Blue-dominated color channels, reduced contrast, haze, and uneven illumination all degrade detection confidence and recall significantly.

By inserting an enhancement stage before detection, the pipeline normalizes each frame toward the visual characteristics the detector was trained to expect — improving both precision and recall without requiring re-training the detector on degraded images.

Without Enhancement

  • Heavy blue-green color cast
  • Low contrast and detail loss
  • High false-negative rate
  • Poor mine detection recall
  • Unreliable confidence scores

With UNet Enhancement

  • Corrected color balance
  • Recovered edges and texture
  • Higher detection confidence
  • Improved mine recall
  • More stable tracking IDs

Model Architecture

METHODOLOGY

The pipeline uses a two-stage approach:

Underwater enhancement stage: A lightweight UNet-based model improves brightness, contrast, and color balance.

Detection stage: The enhanced frame is passed to YOLOv26-M for object detection and tracking.

This design improves both visual quality and detection reliability in underwater scenes.

04
Training Strategy

PROGRESSIVE TRAINING

The UNet was trained across five progressive stages, each introducing additional loss terms to guide the model toward perceptually richer output. This curriculum approach prevents early over-fitting to pixel-level metrics at the expense of structural and perceptual quality.

Training used mixed-precision (FP16 autocast) with AdamW, OneCycleLR scheduling, and weighted sampling to oversample difficult image types — dark, extreme blue, and hazy underwater frames.
